El proyecto “Extracurriculares” es un sistema desarrollado con el objetivo de apoyar a la universidad en la gestión y difusión de sus actividades extracurriculares, facilitando a los estudiantes el acceso a información clara, organizada y actualizada sobre las opciones disponibles fuera del aula.

El sistema permite a los estudiantes consultar las actividades ofrecidas por la universidad, como talleres, deportes, clubes culturales o académicos, mostrando información relevante como el nombre de la actividad, descripción, horario, responsable y disponibilidad. De esta manera, se busca fomentar la participación estudiantil y una mejor organización del tiempo académico y personal.

Desde el punto de vista administrativo, el sistema está pensado para centralizar la información, evitando el uso de medios informales o dispersos (carteles, mensajes o avisos no estructurados), y ofreciendo una plataforma más eficiente y accesible.

Este proyecto tiene un enfoque académico y práctico, ya que integra conceptos de desarrollo de software, organización de información, interfaz amigable y uso de tecnologías web/móviles, aplicando conocimientos adquiridos durante la carrera. Además, busca resolver una necesidad real dentro del entorno universitario, aportando valor tanto a estudiantes como a la institución.
